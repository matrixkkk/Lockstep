# Lockstep
Lockstep synchoronization

# Description

Lockstep 동기화 모델 구현
SceneObject Setting 파일에서 고정 프레임 값과 이벤트가 실행될 턴 시간을 지정할 수 있습니다
기본은 0.033 (30 FPS)로 되어 있으며, 대기해야할 턴 시간은 250 ms 입니다.
Lock된 후에 Unlock 버튼을 통해 다음 턴까지 실행합니다.
